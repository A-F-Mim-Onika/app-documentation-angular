App Color

Steps:

  1. Go to {{codeColor1}} and change the color values to get a light theme.
  2. Go to {{codeColor2}} and change the color values to get a dark theme.

App Font

Steps:

  1. Download a preferred font from the internet. For free Google Fonts click here
  2. Unzip the font.
  3. Go to {{codeFont1}} and paste the unziped font.
  4. Go to {{codeFont2}} and set the font attributese.

    5. For Example:
    family: {{codeFont3}}
    font: {{codeFont4}}
    weight: {{codeFont5}}

  5. Go to {{codeFont1}} , {{codeFont2}} and {{codeFont6}} files and change the name of the font family from there.
