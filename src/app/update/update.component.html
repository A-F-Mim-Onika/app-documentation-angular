Update

Updating from the previous version:

Steps:

  1. Find out the files where the changes have been made after the update.
  2. Usually the name of the folder is like: {{codeUpdate1}} .
  3. Replace the files with the updated one.

Updating after Customisation

There are three ways of updating after customization has been made.

Way 1:

Steps:

  1. Push the customized code on a Git branch
  2. Download the updated code from Codecanyon and push it on a different branch.
  3. Merge both the branches.

Way 2:

Steps:

  1. Find out the files where the changes have been made after the update.
  2. Usually the name of the folder is like: {{codeUpdate1}} .
  3. Merge the updated code with the already customized code.

Way 3:

Steps:

  1. Download the updated code from Codecanyon and repeat the customizations.
