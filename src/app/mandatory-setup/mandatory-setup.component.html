Required Customizations

Steps for Changing App Logo

  1. Go to {{codeLogo1}} and replace logo.png with a different logo.
  2. Go to {{codeLogo1}} and replace logo_name.png with the image file name of the logo.
  3. Go to {{codeLogo2}} and replace all mipmap folder with {{codeLogo3}} folder.
  4. Go to {{codeLogo4}} and replace Assets.xcassets with your generated Assets.xcassets folder.

Steps for Changing App Name

The app name is needed to be changed in three different places.

  1. Go to {{codeName1}} and set the value of {{codeName2}}
  2. Go to {{codeName3}} and change the value of the label.
  3. Go to {{codeName4}} and change the value of CFBundleName .

Steps for Changing Base URL

The admin URL is used as the base URL.

  1. Install Admin Panel.
  2. Go to {{codeUrl3}} and replace the value of the variable {{codeUrl4}} with the URL.

For Example: If the admin URL is {{codeUrl1}} then base URL would be {{codeUrl2}}.
